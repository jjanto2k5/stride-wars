# Stride Wars 🏃‍♂️⚔️

Stride Wars is a location-based mobile application that gamifies running by allowing users to capture and control real-world territories through physical activity[cite: 1].

## 🛠 Tech Stack (MERN)
* **Frontend:** React.js[cite: 1]
* **Backend:** Node.js & Express.js[cite: 1]
* **Database:** MongoDB[cite: 1]

## 🚀 Features
* **GPS Run Tracking:** Track runs and record routes via GPS[cite: 1].
* **Territory Conquest:** Capture new areas by completing running loops and defend them from others[cite: 1].
* **Leaderboards:** Regional and global competitive rankings[cite: 1].

## 💻 Local Development Setup
1. Clone the repository: `git clone <repo-url>`
2. Install dependencies for both frontend and backend using `pnpm install`.
3. Copy `.env.example` to `.env` in both directories and fill in your local MongoDB URI and Map API keys.
4. Start the development servers.
